## 简介

GTest是一个开源的C++单元测试框架，它提供了一系列的断言函数，可以用来对函数的输入输出进行测试。GTest可以运行在Linux、Windows、Mac OS X等平台上。

## 安装

GTest可以从GitHub上下载源码，然后编译安装，不多赘述。下载地址：https://github.com/google/googletest

## 基本用法

### CMake

GTest可以很方便地与CMake集成，只需要在CMakeLists.txt中添加以下代码：

```cmake
cmake_minimum_required(VERSION 3.14)
project(my_project)

# GoogleTest requires at least C++14
# 要求使用C++14，但是我使用C++11，所幸暂时没有用到对应的特性导致项目崩掉，所以先用C++11
set(CMAKE_CXX_STANDARD 11)
set(CMAKE_CXX_STANDARD_REQUIRED ON)

# 从 CMake Modules 目录下找GTest的模块
find_package(GTest REQUIRED)

# 以hello_test作为测试用例
add_executable(hello_test hello_test.cc)
target_link_libraries(hello_test GTest::gtest_main)

enable_testing(False)
add_test("SimpleTest" hello_test)
```

### 编写测试用例

```c++
#include <gtest/gtest.h>

// Demonstrate some basic assertions.
TEST(HelloTest, BasicAssertions) {
  // Expect two strings not to be equal.
  EXPECT_STRNE("hello", "world");
  // Expect equality.
  EXPECT_EQ(7 * 6, 42);
}

TEST(HelloTest, AnotherBasicAssertions) {
  EXPECT_EQ(0, 1);
}
```

这里可以看到gtest提供 TEST 作为基础测试块，按照事件驱动的方式进行测试。
