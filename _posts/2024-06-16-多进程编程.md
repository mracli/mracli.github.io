## 多进程

实现多进程的最简单方式为使用 `fork()` 系统调用：

```c++
#include <sys/types.h>
#include <unistd.h>

pid_t fork();

/*
    成功则创建子进程
    返回：
        子进程返回 0
        父进程返回子进程 pid;
    失败则返回 -1，并设置 errno
*/
```

最终得到相同代码的两个进程，与 fork() 表意相同，即 `分叉`。

### 异同以及影响
相同：父进程与子进程堆指针，栈指针和标志寄存器的值相同；

不同：子进程的 ppid 变为父进程 pid, 信号位图被清除，sighandler 不再对子进程生效。

影响：父进程打开的文件描述符，用户工作目录和当前目录的引用计数 +1; 当父进程或子进程对数据有写操作，则会触发内核的写时复制，为子进程分配内存空间并复制父进程的数据。

### 运行不同的程序

如果需要子进程或父进程在fork后执行不同的代码，除了使用条件判断当前进程，并执行不同代码外：

```C
// 无关代码
pid_t pid;
if(pid = fork() < -1){
    // 创建失败
    exit();
}else if(pid == 0){
    // 父进程
    ...
}else{
    // 子进程
    ...
}
```
